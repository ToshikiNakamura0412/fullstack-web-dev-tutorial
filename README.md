# fullstack-web-dev-tutorial
フルスタックWeb開発を学ぶためのチュートリアル

## 目次（予定）

1. 01-html-css
   - HTML & CSS 基礎（レシピサイトなど）

2. 02-js-basics
   - JavaScript文法基礎（変数・関数・配列・オブジェクト・条件分岐・ループ）
   - 小さなToDoアプリなど

3. 03-js-dom
   - DOM操作 (`document.querySelector`, `classList`, etc.)
   - イベントハンドリング（クリック、フォーム送信など）

4. 04-js-async-and-fetch
   - Promise / async/await
   - fetch API で HTTP 通信（JSON）
   - 公開APIからデータ取得して表示

5. 05-react-basics
   - Reactコンポーネント
   - state / props
   - 簡単なSPA

6. 06-typescript-basics
   - TypeScript基礎（型注釈、オブジェクト、配列、関数の型など）

7. 07-react-with-typescript
   - React + TypeScript のプロジェクト
   - 型付きのコンポーネント設計

8. 08-nextjs-basics
   - Next.js のルーティング
   - ページコンポーネント
   - データ取得（App Router or Pages Router どちらか/両方）

9. 09-node-express-basics
   - Node.js & Express で REST API
   - ミドルウェア / ルーティング
   - 簡単なロギング

10. 10-db-and-orm
    - SQL 基礎（CRUD）
    - PostgreSQL
    - ORM（Prismaなど）でのCRUD

11. 11-logging-and-error-handling
    - バックエンドのロギング
    - エラーハンドリング

12. 12-auth-and-security
    - ユーザー登録 / ログイン
    - パスワードハッシュ
    - JWT 認証

13. 13-realtime-and-websocket
    - WebSocket / Socket.io で簡単なリアルタイム機能

14. 14-deploy-and-infra
    - Vercel / Render / Railway などを使ったデプロイ
    - 環境変数とシークレット管理
